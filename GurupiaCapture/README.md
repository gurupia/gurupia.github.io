# GurupiaCapture GitHub Pages

This directory contains the GitHub Pages deployment for GurupiaCapture.

## Structure

- `index.html` - Main download page
- `releases/` - Release binaries
- `docs/` - Documentation files

## Deployment

1. Build installer: `.\build_installer.ps1`
2. Deploy to gh-pages: `.\deploy_gh_pages.ps1`
3. Push to GitHub: `git push origin gh-pages`
